# odin-recursion
Recursive implementations of a [Fibonacci sequence generator and merge sort](https://www.theodinproject.com/lessons/ruby-recursion), written in Ruby, for [The Odin Project](https://www.theodinproject.com/).

Halfway through the Ruby track, this project has taught or reinforced for me:
- Fundamentals of recursion, including tail recursion to minimize memory usage and space complexity
- Prioritization of efficient vs. simple array manipulation to minimize time complexity
- Implementation of merge sort for improved performance vs. Ruby's built-in quicksort
- Creation of sample scripts for module usage demonstrations

## Pre-requisites for Usage

1. Ruby 3+

## Instructions for Usage

1. Include the Fibonacci Sequence Generator or Sort module in your Ruby code to access its functionality.
2. See the end of the module for examples for calling.

## Instructions for Testing

1. Type `bundle install` into your terminal to install rspec and rubocop.
2. Type `rspec` into your terminal to run all tests contained within the /spec/ folder.

## Rubocop Instructions

1. Type `bundle exec rubocop -a` into your terminal to detect issues.
